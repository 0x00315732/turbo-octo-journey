Коротко:

**Транзакция** — это набор SQL-операций, выполняемых как одно целое.  
Принцип: **все изменения сохраняются, или ни одно не сохраняется**.

Команды:

- **BEGIN** — начать
- **COMMIT** — сохранить
- **ROLLBACK** — отменить

Пример:

```sql
BEGIN;
INSERT INTO table_name VALUES (...);
UPDATE table_name SET ...;
COMMIT; -- или ROLLBACK
```
